# VS Code Manual

整理代码格式

- Windows: `shift + alt + F`
- Ubuntu: `ctrl+ shift + I`

快捷键是可以修改的





打开设置

- `File -> Preference -> Settings`
- `文件 -> 首选项 -> 设置`
- `ctrl + ,`



修改Python格式整理插件 (有`autopep8`, `yapf`, `black`; 目前用的是`black`): `python.formatting.provider`

设置字体: `Font Family`

字体大小: `Font Size`

设置行距: `Line Height`

设置字距: `Letter Spacing`
